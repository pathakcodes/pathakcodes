### Featured projects

#### [learnFromClaude](https://github.com/pathakcodes/learnFromClaude) &nbsp; `2026`

> A Claude Code plugin that turns each session into a stack of learning cards.

Captures every tool call during a session, asks Claude to extract 3–5 declarative learning cards at session end, and surfaces them in a local dashboard with tags, a daily timeline, search, likes, and one-click `claude --resume`. Install with `/plugin marketplace add pathakcodes/learnFromClaude`.

`Claude Code Plugin` · `Hooks` · `Python` · `Vanilla JS` · `Local-first`
